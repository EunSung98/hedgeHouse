# hedgeHouse
- 고슴도치 스마트 케이지 '헷지하우스'의 케이지 코드입니다. 
- C++로 Arduino를 작동했습니다.
- This is the code of 'Hedgehouse',the smart cage for hedgehog.
- It operated Arduino with C++.

### hedgeCage.ino
- 아두이노 메가보드의 코드입니다.
- 주된 기능을 담당하는 메인 보드입니다.

### serial2.ino
- esp32의 코드입니다.
- firebase와 연동합니다.
